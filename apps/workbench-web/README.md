# workbench-web

`workbench-web` is the browser UI for the evidence + semantic workflow.

## Main Stages

- `Evidence`
- `Semantic Spec`
- `Graph`
- `Ask`

The spec stage also includes proposal creation and review, so proposals are no longer a separate primary stage.

## Verify

```bash
npm test --prefix apps/workbench-web
npm run build --prefix apps/workbench-web
```
